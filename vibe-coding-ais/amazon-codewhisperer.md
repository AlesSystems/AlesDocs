# Amazon CodeWhisperer

## Overview
Amazon CodeWhisperer is an AI coding companion from AWS. It provides real-time code suggestions and helps developers build applications faster and more securely.

## Key Features
- **Security Scanning**: Scans code for security vulnerabilities.
- **AWS Service Integration**: Offers code suggestions tailored for AWS services.
- **Reference Tracking**: Flags code suggestions that resemble open-source training data.
- **Multi-language Support**: Supports a wide range of programming languages.

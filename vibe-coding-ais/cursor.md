# Cursor

## Overview
Cursor is an AI-first code editor designed for pair programming with an AI. It is built on top of VS Code and offers a seamless AI integration for a more efficient workflow.

## Key Features
- **Native AI Integration**: AI features are built directly into the editor.
- **Codebase-aware Suggestions**: Provides suggestions based on the entire codebase.
- **Chat with Your Codebase**: Allows you to have a conversation with your code.
- **Automatic Refactoring**: Can automatically refactor code based on your instructions.
